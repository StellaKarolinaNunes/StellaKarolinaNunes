  <h3>✦ Principais Projetos ✦</h3>
   
<h3>
  Meu principal projeto é o <a href="https://github.com/StellaKarolinaNunes/Projeto_Integrador_Handhelp_versao_1.0"> <img src="https://img.shields.io/badge/HandHelp-000000?style=for-the-badge&logo=github&logoColor=white" valign="middle">
  </a>,  um aplicativo móvel assistivo desenvolvido no Projeto Integrador para auxiliar pessoas surdas durante o atendimento de triagem em Unidades Básicas de Saúde. A proposta une tecnologia, acessibilidade, saúde digital e apoio visual em Libras, tornando a comunicação mais clara, inclusiva e eficiente. Também desenvolvi o <a href="https://github.com/StellaKarolinaNunes/HandBits"> <img src="https://img.shields.io/badge/HandBits-000000?style=for-the-badge&logo=github&logoColor=white" valign="middle"> </a>, um projeto interativo voltado para educação e tecnologia, utilizando visão computacional para ensinar contagem binária por meio de gestos manuais. Ele combina aprendizado, gamificação e conceitos de inteligência computacional de forma prática e criativa. Outro projeto de destaque é o <a href="https://github.com/StellaKarolinaNunes/NadoFit"> <img src="https://img.shields.io/badge/NadoFit-000000?style=for-the-badge&logo=github&logoColor=white" valign="middle"> </a>, uma plataforma web/mobile voltada para escolas de natação, criada com foco em interface moderna, experiência do usuário, responsividade e design visual premium. O projeto destaca meu interesse por UX/UI, Flutter e desenvolvimento de interfaces funcionais. O <a href="https://github.com/StellaKarolinaNunes/VinculoTEA"> <img src="https://img.shields.io/badge/VinculoTEA-000000?style=for-the-badge&logo=github&logoColor=white" valign="middle">
  </a>  é uma plataforma de gestão integrada para acompanhamento de alunos com TEA, reunindo funcionalidades como dashboard, agenda, gestão de disciplinas, anamnese, relatórios e apoio ao Plano de Ensino Individualizado. A proposta busca contribuir com organização, inclusão e acompanhamento educacional. Veja meus outros projetos <a href="https://github.com/StellaKarolinaNunes?tab=repositories"><img src="https://img.shields.io/badge/Ver%20todos-000000?style=for-the-badge&logo=github&ogoColor=white" /> </a>
